## Project Description
E-Masjid System is a web-based mosque management platform designed for Pakistani mosques. It provides:
- Transparent donation management
- Prayer time display
- Event management
- Online nikah registrar booking 
- Announcement system

## Technology Stack
- **Frontend:** React.js
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Authentication:** JWT
